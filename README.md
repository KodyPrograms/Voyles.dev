# Voyles.dev

Personal portfolio site for [voyles.dev](https://voyles.dev), built to present the kind of work I want to keep doing: internal tools, automation, reporting systems, production web work, and software tied to real business needs.

The site is built with React, TypeScript, Vite, and Tailwind CSS, and is intended to be lightweight, fast to deploy, and easy to maintain.

## What This Site Covers

- Internal platforms and operational software
- Reporting, analytics, and automation-focused projects
- Production website and marketing technology work
- Public project links, demos, and GitHub repositories

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Framer Motion

## Local Development

Install dependencies:

```bash
npm install
```

Start the local dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

This project is configured to deploy cleanly on Cloudflare Pages.

- Build command: `npm run build`
- Output directory: `dist`

## Repository Notes

This repository is for the public portfolio site only. Individual project repositories, live demos, and production examples are linked from the site itself where appropriate.
